Chandra – Wisteria

Statement: “Wisteria kills the tree it climbs on because it feeds on it.”

Logical Proposition: W: If wisteria climbs a tree, then it feeds on the tree.
Symbolized: (W → F)

Truth Value: ❌ False

Scientific Explanation: Wisteria is not a parasite; it doesn’t get nutrients from the host tree. It harms trees by wrapping tightly and shading, which can lead to damage over time. It absorbs nutrients from the soil instead.

Ashley – Tulip

Statement: “Tulips bloom only during spring.”

Logical Proposition: T: If it is a tulip, then it blooms only in spring.
Symbolized: (T → B)

Truth Value: ✅ True

Scientific Explanation: Tulips are perennials that bloom once a year during spring, needing cold dormancy to flower. After blooming, they enter dormancy again for the rest of the year.

Sources:
Royal Horticultural Society
University of Illinois Extension
Old Farmer’s Almanac
The Garden Philippines (Local)

Kiesha – Orchid

Statement: “All orchids grow only on trees.”

Logical Proposition: O: If it is an orchid, then it grows on trees.
Symbolized: (O → T)

Truth Value: ❌ False

Scientific Explanation: Not all orchids grow on trees. Some are terrestrial (grow in soil), while others grow on rocks. Orchids are one of the most diverse plant families, with species adapted to many environments.

Sources:
American Orchid Society
Britannica
Royal Botanic Gardens, Kew
PNPCSI (Local)
